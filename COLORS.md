# Studio—E Color System

## Palette

| Name       | Hex       | Role                        |
|------------|-----------|-----------------------------|
| Ink        | `#151514` | Dark background             |
| Charcoal   | `#3A3A37` | Secondary text              |
| White      | `#F7F5F2` | Primary text on dark bg     |
| Parchment  | `#E6E6DB` | Light background            |
| Periwinkle | `#D0D1FF` | Soft accent, tags, surfaces |
| Olive      | `#B2B41F` | CTA & accent                |
| Violet     | `#422DA2` | Eyebrow / label text        |

## Color Roles

- **Ink `#151514`** — Primary background in dark mode. Pairs with White or Parchment text.
- **Parchment `#E6E6DB`** — Primary background in light mode. Pairs with Ink text.
- **Olive `#B2B41F`** — Buttons, highlights, active states. Used on both dark and light backgrounds. Always paired with Ink text.
- **Violet `#422DA2`** — Small labels and eyebrow text on light backgrounds only.
- **Periwinkle `#D0D1FF`** — Subtle highlights, tags, tonal surfaces. Never use for text.
- **Charcoal `#3A3A37`** — Body text on light backgrounds. Secondary UI elements.

## Dark Mode

- Background: Ink `#151514`
- Headline: White `#F7F5F2`
- Body: `#888888`
- Eyebrow: Olive `#B2B41F`
- CTA button: Olive `#B2B41F` with Ink `#151514` text

## Light Mode

- Background: Parchment `#E6E6DB`
- Headline: Ink `#151514`
- Body: Charcoal `#3A3A37`
- Eyebrow: Violet `#422DA2`
- CTA button: Olive `#B2B41F` with Ink `#151514` text

## Approved Pairings

| Pairing                  | Use Case                        |
|--------------------------|---------------------------------|
| Ink + White              | Primary text on dark bg         |
| Parchment + Ink          | Primary text on light bg        |
| Olive + Ink              | CTA buttons, both modes         |
| Parchment + Violet       | Eyebrow / label text, light only|
| Ink + Olive              | Accent on dark bg               |
| Parchment + Charcoal     | Body copy on light bg           |
